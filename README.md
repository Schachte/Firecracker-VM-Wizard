# Firecracker MicroVM Wizard

Simplifying the Firecracker VM setup process. Inspired by [gruchalski](https://gruchalski.com/).

# Usage

```sh
git clone git@github.com:Schachte/Firecracker-VM-Helper.git && \
cd Firecracker-VM-Helper && \
./wizard
```
